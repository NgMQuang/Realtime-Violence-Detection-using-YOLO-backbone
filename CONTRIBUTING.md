# Contributing

Thanks for wanting to help! Here's how you can contribute:

## Issues & Bugs

Found a bug? Open an issue with:
- What's broken
- How to reproduce it
- What you expected to happen

## Feature Ideas

Got an idea? Open an issue describing:
- What you want to add
- Why it would be useful

## Code Contributions

### Setup

1. Fork and clone the repo
2. Install deps: `pip install ultralytics`. Yet that's all
3. Create a branch: `git checkout -b my-changes`

### Making Changes

- Keep it simple and focused
- Write clear commit messages
- Test your changes locally

### Submitting

1. Push your branch
2. Open a pull request with a description of what you changed
3. I'll review and merge if it looks good

### Code Style

- Use 2-space indentation
- Use meaningful names for variables/functions
- Add comments for complex logic

## That's It!

No complicated processes here. Just make useful changes and submit them. Thanks! 🎉
